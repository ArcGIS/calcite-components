# Custom theme

Developers building Esri applications are encouraged to use the default light or dark themes as much as possible. However, [CSS variables](#available-css-varirables) are exposed on `:root` and can be used to customize [color](#color-vars) and [typography](#typography-vars), and in one case, [spacing](#panel-width-var).

This example demonstrates changing two colors.

```html
<style>
  .my-theme {
    --calcite-ui-text-3: white;
    --calcite-ui-foreground-1: red;
  }
</style>
<div class="my-theme">
  <calcite-action icon="beaker" text="Foo" text-enabled></calcite-action>
</div>
```

CalciteAction uses `--calcite-ui-foreground-1` and `--calcite-ui-text-3` for its default appearance. Overwriting thosee variable will customize the appearance of all instances of CalciteAction inside the div.

Setting a valid theme CSS class on individual components or elements will prevent your overwrites from taking effect.

```html
<style>
  .my-theme {
    --calcite-ui-text-3: white;
    --calcite-ui-foreground-1: red;
  }
</style>
<div class="my-theme">
    <calcite-action icon="beaker" text="Foo" text-enabled></calcite-action>

    <!-- The valid theme class will prevent this from recieving the overwrites. -->
    <calcite-action class="light" icon="beaker" text="Bar" text-enabled></calcite-action>
</div>
```

<h3 id="panel-width-var">Panel width multiplier</h3>
Applications regularly use CalciteShell and its child components (CalciteShellPanel, CalcitePanel, etc.) for layout. The width of the panels in this framework is dynamic based on a combination of a predfined ratio and the `width-scale` property on CalciteShellPanel and CalcitePanel. In order to customize that ratio, use the `--calcite-panel-width-multiplier`.

This `--calcite-panel-width-multiplier` variable's default value is 1. Setting it to greater to or less than that will have an application-wide effect.

```html
<style>
:host {
  --calcite-panel-width-multiplier: 1.25; // Increases the width by 25%.
}
</style>
```

```html
<style>
:host {
  --calcite-panel-width-multiplier: .75; // Descreases the width by 25%.
}
</style>
```
